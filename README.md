#Generations

**config:**
*(see config.txt)*
```
    Num of Factions -> 1 to 8
    fac1 -> Color, (minStr, maxStr) (minRepr, maxRepr) (x, y)
    ...
    facN -> Color (minStr, maxStr) (minRepr, maxRepr) (x, y)
    mutation -> (-1.0, 1.0)
    mutationChance -> 0.0 to 1.0
    sickness -> 1:YES | 0:NO
```


**COLORS:**

RED = (255,0,0)
BLUE = (0,0,255)
GREEN = (0,255,0)
YELLOW = (255,255,0)
MAGENTA = (255,0,255)
CYAN = (0,255,255)
ORANGE = (255,165,0)
PURPLE = (128,0,128)
WHITE = (255,255,255)
BLACK = (0,0,0)
